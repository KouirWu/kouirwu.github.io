49. 字母异位词分组

题目链接：49. 字母异位词分组

题目描述

给你一个字符串数组 strs，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

题目要求

给定一个字符串数组 strs，将字母异位词（Anagram）组合在一起，并返回这些组合后的列表。

字母异位词（Anagram）定义

  • 定义：两个字符串如果包含相同的字母（顺序无关），则互为字母异位词。
  • 示例：
    • "eat" 和 "tea" 是字母异位词，因为它们的字母顺序不同但组成相同。
    • "rat" 和 "bat" 不是字母异位词，因为它们包含的字母不同。

输入输出示例

输入：strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出：

解题思路

核心思想：

  • 字母异位词在排序后会变成相同的字符串

    • "eat"、"tea"、"ate" 排序后都变成 "aet"
    • "tan"、"nat" 排序后变成 "ant"
    • "bat" 保持不变 "abt"（"bat" 是唯一一个）

  • 利用哈希表（字典）分组存储：

    • 使用 排序后的字符串 作为 key
    • 相同 key 的字符串分组存储

步骤：

  1. 初始化一个 defaultdict(list)，用于存储分组结果
    (defaultdict 可以避免 KeyError，如果 key 不存在会自动创建空列表)
  2. 遍历每个字符串 s：
    • 排序 s 得到一个标准化的 key
    • 把 s 添加到 d[key] 的分组里
  3. 最后返回字典的所有 values()，就是分组后的结果

代码详细解释

逐行解析：

  1. d = defaultdict(list)

    • 初始化为一个空字典，键 (key) 是排序后的字符串，值 (value) 是一个列表（用于存同一分组的所有字符串）。

  2. for s in strs

    • 遍历输入列表 strs 里的每一个字符串 s。

  3. sorted_s = ''.join(sorted(s))

    • sorted(s) 把字符串 s 转换成排序后的字符列表，例如：
      • sorted("tea") → ['a', 'e', 't']
      • ''.join(['a', 'e', 't']) → "aet"（还原成一个标准化的字符串）
    • 为什么排序？
      → 因为异位词排序后会一致，可以直接比较！
      → 比如 "eat"、"tea"、"ate" 都会变成 "aet"，它们就可以被正确分组。

  4. d[sorted_s].append(s)

    • 把原始字符串 s 存进对应的分组里
      • 如果 sorted_s 已经存在，则直接 append。
      • 如果 sorted_s 不存在，defaultdict 自动初始化一个空列表，再添加 s。

  5. return list(d.values())

    • d.values() 返回所有分组（是一个列表的集合），但需要转换成 list 类型（Python 3 中 dict_values 不是直接可索引的）。

时间复杂度分析

  • 排序每个字符串 🔄 → O(N * K log K) （其中 N 是 strs 的长度，K 是最大字符串长度）
  • 哈希表查存操作 🗃️ → O(N)（每个字符串存一次）
  • 总时间复杂度：O(N * K log K)
    （优于 O(N^2) 的暴力解法）

对比其他方法

方法 时间复杂度 空间复杂度 适用场景
排序 + 哈希表（当前方法） O(N * K log K) O(N * K) 最优解
暴力法（比较每个字符串） O(N^2 * K) O(N * K) 不推荐（太慢）
使用字符计数作为 key O(N * K) O(N * K) 适用于字符集较小

总结

  1. 排序 + 哈希表 是解决字母异位词分组的最优方法之一。（O(N * K log K)）
  2. defaultdict(list) 可以方便地存储分组。
  3. 排序后的字符串作为 key，保证字母异位词一定属于同一分组。
  4. 代码简洁高效，面试常考题，需要熟练掌握！

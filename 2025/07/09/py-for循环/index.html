🚀 精通 Python for 循环：从入门到高效实践

Python 的 for 循环是其最强大、最灵活的特性之一。如果你有其他编程语言的背景，你可能会习惯于 C 风格的 for (int i = 0; i < 5; i++) 循环。然而，Python 的 for 循环在设计上更接近“for-each”循环，它直接遍历集合中的每一个元素，这使得代码更加简洁和易读。

在这篇博文中，我们将深入探讨 Python for 循环的各种用法，从基础的元素遍历，到使用 range(), enumerate() 等内置函数的高效实践。

1. 基础用法：直接遍历可迭代对象

这是 for 循环最核心、最直观的用法。它可以直接遍历任何“可迭代对象”（如列表 list, 元组 tuple, 字符串 str, 集合 set 等）中的每一个元素。

2. range() 函数：按次数或索引循环

当你需要循环一个固定的次数，或者需要根据索引来访问元素时，range() 函数就派上用场了。它能生成一个不可变的整数序列。

range() 主要有三种形式：

a. range(stop)

生成一个从 0 开始到 stop - 1 的整数序列。

输出：

循环次数: 0
循环次数: 1
循环次数: 2
循环次数: 3
循环次数: 4

b. range(start, stop)

生成一个从 start 开始到 stop - 1 的整数序列。

输出：

序列中的数字: 2
序列中的数字: 3
序列中的数字: 4
序列中的数字: 5

c. range(start, stop, step)

生成一个从 start 开始，每次按 step 步长递增，直到 stop 的序列。

输出：

奇数: 1
奇数: 3
奇数: 5
奇数: 7
奇数: 9

虽然你可以使用 range(len(my_list)) 来通过索引遍历列表，但这通常被认为是不够 “Pythonic” 的做法。对于这种情况，我们有更好的工具！

a. zip() - 并行遍历

使用 zip() 函数可以“压缩”多个可迭代对象，将它们对应位置的元素打包成元组，然后进行并行遍历。当最短的那个可迭代对象耗尽时，循环就会停止。
